# Checkverse Marketing Website

Marketing website for Checkverse - AI-powered claims auditing platform for insurance companies.

## Overview

A single-page marketing site targeting insurance company decision makers. Built with plain HTML and Tailwind CSS.

## Structure

```
checkverse-website/
├── index.html    # Main marketing page
├── favicon.svg   # Brand icon
└── README.md     # This file
```

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000`

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- No build process required
